App for iOS

I'm currently working on an app for iOS (and macOS), so you can have widgets on your homescreen to show the next concert.

Features

  • Widgets with a countdown to a Linkin Park show. Either...
    • ... next concert
    • ... next concert you have bookmarked
    • ... next concert you are planning to attend
  • List of all upcoming concerts
  • bookmark concerts
  • Filter for concerts by tour and date
  • Map of all upcoming concerts (and past concerts since 2024)

Download
