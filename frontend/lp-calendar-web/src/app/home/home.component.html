LP Concert Calendar

Want too keep track of the From Zero World Tour? Subscribe to the iCal-feed that contains all the concerts, so you never miss a show.

Subscribe to calendar feed

To subscribe to the calendar, you need to use the following URL: {{environment.apiCachedBaseUrl}}/Prod/feed/ical
You can enter this URL in most calendar applications. The application will periodically refresh the calendar to get any changes in the schedule.

View concert list

You can check out the list of all concerts we currently track on this website. Please note that this is maintained by fans and is not official information!
