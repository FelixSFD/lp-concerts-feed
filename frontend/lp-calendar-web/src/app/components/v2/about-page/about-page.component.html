About LPshows.live

This page is a project made by Linkin Park fans to keep track of all the shows in different timezones. With almost 100 shows around the world for the FROM ZERO WORLD TOUR, things can get confusing. That's why this project shows you all the shows in your local and also the venue's timezone.

Over time, some more features were added to more easily share the times.

Calendar feed

Many calendar applications (like on iOS, iPadOS and macOS) support calendar subscription. This project publishes a calendar feed with all the shows that you can use in your calendar app.

FAQ

Who is behind this project?

This project was started by FelixSFD to learn new technologies and get an overview of all shows in his local timezone.

 Do I need to create an account?

If you want to bookmark concerts or link your account to a concert that you attend, you will have to create an account. But even without an account, you can access all the information about the concerts.

 How can I contribute?

In the future, select users might be added as administrators to manage the concert list. You can find the code on GitHub, but at the moment, there are no contributions needed as there is not a clear plan for the project yet.

 Can I use the data?

Yes! The data is supposed to be public. If you have questions about the APIs, feel free to contact FelixSFD on Discord.

Please tell FelixSFD if you rely on the APIs for your project. Otherwise the API might change without you knowing about it and break your application.

 Can I get access to the statistics about the tour?

While the data is intended to be public, making the full dashboard available to the public could cause a lot of load on the database server. That's why only select users can get access to it.

If you need access for your fan-page (or something similar), please contact FelixSFD on Discord.

 Where does the data for the map come from?

Each location on the map was manually selected and saved to our database. The map itself is from Apple Maps.

 Where can I find the history of past concerts?

At the moment, this page is intended for future concerts. If you are interested in past concerts, setlists and more details about the shows, you should check out LPLive.net. They have a great database with detailed information about every single Linkin Park show.

Right now, our page has information about all concerts since the comeback in 2024.
