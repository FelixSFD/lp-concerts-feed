About this page

This page is a project made by Linkin Park fans to keep track of all the shows in different timezones. With over 60 shows around the world in 2025, things can get confusing. That's why this project shows you all the shows in your local and also the venue's timezone.

Calendar feed

Many calendar applications (like on iOS, iPadOS and macOS) support calendar subscription. This project publishes a calendar feed with all the shows that you can use in your calendar app.



FAQ

Who is behind this project?

This project was started by FelixSFD to learn new technologies and get an overview of all shows in his local timezone.

Do I need to create an account?

No. At the moment, the accounts are only used by administrators to manage the concert data.

How can I contribute?

At the moment, the page is still in early development.

In the future, select users might be added as administrators to manage the concert list. You can find the code on GitHub, but at the moment, there are no contributions needed as there is not a clear plan for the project yet.

Can I use the data?

Yes! The data is supposed to be public. If you have questions about the APIs (warning, they might change as they are in early development!), feel free to contact FelixSFD on Discord.

How does this work?

The frontend is hosted as an Angular single-page-application in an AWS S3 bucket behind a CloudFront distribution for caching and having a custom domain.

Backend logic and the database is hosted in AWS. The database is a DynamoDB while the logic for reading and writing is implemented in AWS Lambda functions running .NET 8. The API is cached by AWS CloudFront.

To generate the calendar feed, one API endpoint creates an iCal file that contains all the concerts using the iCal.NET library. When you subscribe to the calendar, your calendar application contacts our server regularly to get an updated file.

Where does the data for the map come from?

Each location on the map was manually selected and saved to our database. The map itself is from OpenStreetMap.

Where can I find the history of past concerts?

This page is intended for future concerts. If you are interested in past concerts, setlists and more details about the shows, you should check out LPLive.net. They have a great database with detailed information about every single Linkin Park show.
